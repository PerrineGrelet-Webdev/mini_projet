# Projet React – Générateur dynamique de besoins en protéines

  ## Présentation rapide
  
Cette application génère un tableau des besoins journaliers en protéines en fonction :

- du poids (min, max, nombre de valeurs)

- des objectifs sportifs choisis
  (sédentaire, endurance, maintien musculaire, prise de masse)

## Installation & lancement (npm)
  ### Prérequis
  
Node.js installé

Avant de lancer le projet, il faut avoir Node.js installé sur son ordinateur.
Pour l'installer, if faut :

- Aller sur : https://nodejs.org
- télécharger la version recommandée
- L'installer comme n’importe quel logiciel (npm sera automatiquement installé avec Node)


  ### Installer les dépendances
  
Dans le terminal, de préférence wsl, on se place dans le dossier où est enregistré le projet :

**cd /mnt/c/Users/*/Documents/*/*/mini_projet**

Puis installe les dépendances :

**npm install**

### Lancer le projet

**npm run dev**

L’application sera accessible via le lien affiché dans le terminal.
